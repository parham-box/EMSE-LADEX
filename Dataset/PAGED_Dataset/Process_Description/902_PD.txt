For EC-Lille: Initially, if the option "research schedule directly" is selected, then the action to be taken is "Research schedule". If the option is not selected, then "provide indications" and proceed with the research schedule. After validating the schedule, check if the schedule is satisfactory. If it is not, then perform the action "Research schedule". If it is, then proceed with writing a ticket order to SNCF. If the schedule is not satisfactory, perform the action "Research schedule". If it is still not satisfactory, then "provide indications". Alternatively, if the schedule is satisfactory, write a ticket order to SNCF. After writing the ticket order, validate it. Check if the ticket order is approved. If it is not, then the process ends. If it is approved, then validate the ticket order. Check again if the ticket order is approved. If it is not, then the process ends. If it is approved, then validate the ticket order. Check once more if the ticket order is approved. If it is, then the process ends. If it is not, then the process ends.
